Event-study для Российского рынка

В данной работе представлен инструмент для проведения событийного анализа, который позволяет быстро и эффективно проводить событийный анализ на любом рынке и для любого набора акций. С помощью данного инструмента были проанализированы 4 новостных шока на российском рынке: ввод войск на Украину в марте 2014 года, девальвация рубля в декабре 2014 года, арест Владимира Евтушенкова в сентябре 2014 года и слухи о банкротстве «Мечела» в ноябре 2013 года. Было проанализировано поведение 24 акций из 8 индустрий.
